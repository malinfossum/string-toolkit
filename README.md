# string-toolkit

Pair programming exercise (Week 2 of Emne 3 at Get Academy). Console app that reverses a string without using built-in reverse methods, plus a menu for uppercase/lowercase conversion. Built with [@CozREV](https://github.com/CozREV).

## Requirements

.NET 8 SDK or later — [download from Microsoft](https://dotnet.microsoft.com/download).

## Run

```powershell
dotnet run
```
